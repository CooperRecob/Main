Create an online order form for an online computer store and include the following items: Drop-down menu for selecting an operating system. Include at least Windows and Linux as the options. Drop-down menu for selecting a processor. Include Intel Core i5, Intel Core i7, and AMD Ryzen as the options. Intel Core i5 should be the default value. Group of radio buttons for selecting memory. 8 GB should be the default value. Checkboxes for selecting accessories, as in the image. These are optional items. Text elements for entering name and email address. Text area to enter comments. submit button to send form data to a server and cancel button to clear the selections Note: the submit button is just to create a complete form. You do not have to define the form action attribute for this assignment. Online Order Form
